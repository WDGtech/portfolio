Project 3: Slagboom Opstelling met ESP32

Dit project zorgt voor de beweging van de slagboom door een servomotor. Het systeem is volledig geprogrammeerd in de Arduino IDE en verbonden via een breadboard. Dit project biedt inzicht in hardware-integratie en de combinatie van software en elektronica voor het bouwen van een functioneel en gesynchroniseerd systeem.

Project 4: Netwerkmonitoring en Beveiliging

Dit project is een theoretische denkoefening gericht op het opzetten van een netwerkmonitoringssysteem met SNMP en Syslog. Het doel is om netwerkprestaties en beveiligingsproblemen te bewaken, waarschuwingen te genereren en een visueel dashboard te ontwerpen voor real-time inzicht.

Project 5: Datacenter Ontwerp

Dit project richt zich op het ontwerpen van een virtueel datacenter met behulp van tools zoals VMware, Visio en FreeNAS. Het doel is om netwerksegmentatie, virtuele serverconfiguratie en opslagbeheer te plannen. Door een visueel stappenplan te maken, worden belangrijke concepten zoals VLANs, back-ups en schaalbaarheid verkend, zonder praktische uitvoering.

Project 6: Netwerk Scanner met Python

Dit project richt zich op het ontwikkelen van een eenvoudig netwerk scanning-tool met Python. Het is gericht op netwerk- en systeembeheer en helpt om inzicht te krijgen in de apparaten die op een lokaal netwerk actief zijn. Dit is nuttig om ongewenste apparaten te detecteren of om netwerkconfiguraties te controleren.
